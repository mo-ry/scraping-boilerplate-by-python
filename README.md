# scraping-boilerplate-by-python

## Introduction
This is a scraping boilerplate by python3 which provides you a basis of scraping a website with authentication and saving scraped items in a database.

## Prerequisite
- Docker

## How to use
- Add your `.python.env` and `.docker-compose.env` in reference to sample files.
- Customize `scrape.py` to scrape and save items you want.
- Run `docker-compose up` in this project directory
- Then, you have items you want in the database.
